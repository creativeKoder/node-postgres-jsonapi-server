# Nodejs PostgreSQL API Server

This is a boilerplate API server using Node.js and PostgreSQL.  It has a TODO example and user auth (which needs some work to finish up)

## Requirements
Node.js (duh) Preferably 4.x.x +

PostrgreSQL 9.4+

Nodemon
